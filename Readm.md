# Mavano Sport Kids Academy Website
The official website of Mavano Sports Academy, a professional platform that connects high-quality sports coaches with preschools to promote early-age physical development among children. The website provides information about the academy’s services, programs, and contact details in a clean, modern, and responsive design.

Features:

Responsive and mobile-friendly layout

Contact form and WhatsApp chat integration

Clear call-to-action buttons (Join Now, Contact Us)

Custom domain and professional email integration

SEO-friendly structure with fast loading performance

Tech Stack:

HTML5

CSS3 (Bootstrap 5)

JavaScript

Hosted on Vercel

Domain managed via GoDaddy

Business email powered by Zoho Mail

Purpose:
To serve as the official online presence for Mavano Sports Academy, providing an easy and professional way for preschools and parents to connect with certified coaches.
 
## Overview
A responsive, modern landing page for Mavano Sport Kids Academy.  
Built with Bootstrap 5, HTML, CSS, and JavaScript.

## Structure
- `index.html` → Main page
- `css/style.css` → Custom styles
- `js/script.js` → JavaScript functionality
- `img/` → Images
- `README.md` → Project documentation

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
3. For deployment, upload the folder to GitHub Pages / Netlify / Vercel
